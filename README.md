<!-- GETTING STARTED -->
## Getting Started

Start by by installing the packages and going to localhost to play the games.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/shawncrook411/minigame.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
